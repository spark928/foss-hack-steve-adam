# Study-Flow
StudyFlow is a structured study productivity app that organizes learning by subjects and chapters. It includes notes, flashcards, mind maps, study timers, and task tools. The app tracks study time and visualizes weekly/monthly performance with graphs to help students monitor progress and improve productivity.
